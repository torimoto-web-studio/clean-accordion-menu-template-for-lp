# Clean Accordion Menu Template for LP

LPや下層ページ案件で使える、シンプルなアコーディオンUIのテンプレートです。

## Overview
- JavaScriptでクラスの付け外しのみを行うシンプルな設計
- SCSSで表示・非表示を制御
- テンプレートとして再利用しやすい構成

## File Structure

clean-accordion-menu-template-for-lp/
├─ index.html
├─ css/
│ └─ style.css
├─ scss/
│ └─ style.scss
├─ js/
│ └─ script.js

## Features
- クリックした項目のみ開くアコーディオン
- 複数設置を想定した構成
- LP案件での使い回しを想定

## Purpose
実案件で使えるUIテンプレートを作ることを目的に制作しました。  
今後、アクセシビリティや拡張性の改善を予定しています。
